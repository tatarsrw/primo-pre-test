โจทย์ : For implementing function with this interface merge (int[] collection_1, int[] collection_2, int[] collection_3) : int []
เงื่อนไข :
1. collection_1, collection_3 already sorted from min(0) to max
2. collection_2 already sorted from max to min(0)
3. Don't allow to use any sort function*
กำหนดตัวแปร :
collection_1 = [5, 2, 4, 6, 0, 3, 1];
collection_2 = [9, 12, 7, 10, 11, 8];
collection_3 = [14, 13, 16, 17, 15, 18];
